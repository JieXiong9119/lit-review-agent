# `projects/_example/` — worked reference deliverable

This folder is the agent's "definition of done" — a complete, real (small)
literature review showing every output file the workflow produces, end to end.

**Topic:** Retrieval-Augmented Generation and the hallucination frontier (3
papers, 2023–2024, ~330-word draft).

**Why so small?** The example is meant to be readable in one sitting. A real
review will be 5–10× larger in catalog size and word count.

## Caveats

- All three catalog entries have `status: metadata-only`. The example does not
  ship third-party PDFs (the gitignore allows PDFs here, but we keep the
  worked example clean of redistribution concerns by working from abstracts
  only).
- The draft prose is deliberately brief and based on what the abstracts of the
  three papers actually say. It is not a substitute for reading the full
  papers.

## Regenerating `references.bib`

From the repo root:

```powershell
python tools/build_bib.py --project projects/_example `
    --only-cited projects/_example/lit-review.md `
    --out projects/_example/references.bib
```
